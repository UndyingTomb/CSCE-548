Pokemon Card Tracker — Project 3 Client

This client calls GET endpoints for all tables: get all, get one, and get subset (filter).

Client runs at http://127.0.0.1:5500, API runs at http://127.0.0.1:8000.

Conditions (GET all, GET one, GET subset)

GET /conditions (all)

GET /conditions/{id} (single)

GET /conditions?query=... (subset)

Sets (GET all, GET one, GET subset)

GET /sets (all)

GET /sets/{id} (single)

GET /sets?set_code=... (subset)

Cards (GET all, GET one, GET subset)

GET /cards (all)

GET /cards/{id} (single)

GET /cards?set_id=...&rarity=... (subset)

Subset demo: filter by set + rarity.

Inventory (GET all, GET one, GET subset)

GET /inventory (all)

GET /inventory/{id} (single)

GET /inventory?is_graded=... (subset)

Subset demo: show only graded or only ungraded items.
